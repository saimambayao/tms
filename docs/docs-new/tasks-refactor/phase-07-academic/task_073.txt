Create new Django app for academic records management including grades, attendance, and academic calendar functionality.

## Implementation Steps:
1. Create academics app:
   ```bash
   cd /Users/saidamenmambayao/apps/madaris-ms/src/apps
   python ../manage.py startapp academics
   ```
2. Add to INSTALLED_APPS in settings:
   ```python
   # config/settings/base.py
   INSTALLED_APPS = [
       ...
       'apps.students',
       'apps.academics',  # New app
       ...
   ]
   ```
3. Create __init__.py files:
   ```bash
   touch /Users/saidamenmambayao/apps/madaris-ms/src/apps/academics/__init__.py
   ```
4. Run initial migration:
   ```bash
   python manage.py makemigrations academics
   python manage.py migrate academics
   ```
5. Verify in Django admin
