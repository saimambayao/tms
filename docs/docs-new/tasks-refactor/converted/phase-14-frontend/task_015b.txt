# Details:
Extend existing Django REST Framework API endpoints to support Next.js frontend. Keep changes minimal - only add what's needed.

## Implementation Steps:
1. Install additional DRF packages:
   ```bash
   pip install djangorestframework-simplejwt django-cors-headers
   pip freeze > requirements.txt
   ```
2. Update settings.py:
   ```python
   INSTALLED_APPS += ['corsheaders']
   MIDDLEWARE.insert(0, 'corsheaders.middleware.CorsMiddleware')
   CORS_ALLOWED_ORIGINS = ['http://localhost:3000']  # Next.js dev server
   ```
3. Configure JWT authentication in settings.py:
   ```python
   REST_FRAMEWORK = {
       'DEFAULT_AUTHENTICATION_CLASSES': [
           'rest_framework_simplejwt.authentication.JWTAuthentication',
       ],
   }
   ```
4. Add JWT token endpoints to urls.py:
   ```python
   from rest_framework_simplejwt.views import TokenObtainPairView, TokenRefreshView

   urlpatterns += [
       path('api/token/', TokenObtainPairView.as_view()),
       path('api/token/refresh/', TokenRefreshView.as_view()),
   ]
   ```
5. Test API endpoints with curl or Postman
6. Document API endpoints
