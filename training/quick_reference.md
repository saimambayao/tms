# #FahanieCares Quick Reference Guide

## Common Tasks

### Login
1. Go to website
2. Click "Login"
3. Enter username/password
4. Click "Sign In"

### Create New Constituent
1. Menu → Constituents → Add New
2. Fill required fields
3. Click "Save"

### Create New Referral
1. Menu → Referrals → Create New
2. Select constituent
3. Choose service
4. Add details
5. Click "Submit"

### Search
1. Use top search bar
2. Enter keywords
3. Press Enter
4. Filter results as needed

### Generate Report
1. Menu → Reports
2. Select report type
3. Set date range
4. Click "Generate"

## Status Codes

### Referral Status
- **Pending**: Awaiting action
- **Processing**: Being worked on
- **Completed**: Finished
- **Cancelled**: No longer needed

### Priority Levels
- **Urgent**: Immediate action
- **High**: Within 24 hours
- **Normal**: Within 3 days
- **Low**: Within 1 week

## File Upload Limits

- **Max Size**: 10MB per file
- **Formats**: PDF, JPG, PNG, DOC, DOCX
- **Multiple Files**: Yes (up to 5 at once)

## Common Issues

### Can't Login?
1. Check username/password
2. Clear browser cache
3. Try different browser
4. Contact IT support

### File Won't Upload?
1. Check file size (<10MB)
2. Verify file format
3. Check internet connection
4. Try smaller file

### Report Not Loading?
1. Reduce date range
2. Remove some filters
3. Try different format
4. Export instead
