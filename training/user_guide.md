## Getting Started

### Logging In

1. Navigate to the #FahanieCares website
2. Click "Login" in the top right corner
3. Enter your username and password
4. Click "Sign In"

### Dashboard Overview

After logging in, you'll see your dashboard with:
- Recent activities
- Pending tasks
- Quick actions
- Key metrics

### User Roles

- **Admin**: Full system access
- **Staff**: Manage constituents and referrals
- **Chapter Coordinator**: Manage chapter activities
- **Public User**: Submit requests and view status

## Constituent Management

### Adding a New Constituent

1. Click "Constituents" in the main menu
2. Click "Add New Constituent"
3. Fill in the required information:
   - Full Name
   - Contact Number
   - Email Address
   - Address
   - Municipality
   - Barangay
4. Click "Save"

### Searching for Constituents

1. Use the search bar at the top
2. Enter name, contact number, or address
3. Use filters for advanced search:
   - Municipality
   - Barangay
   - Chapter membership
   - Age range

### Updating Constituent Information

1. Find the constituent using search
2. Click on their name
3. Click "Edit"
4. Update the information
5. Click "Save Changes"

## Service Referrals

### Creating a New Referral

1. Click "Referrals" in the main menu
2. Click "Create New Referral"
3. Select or create constituent
4. Choose service category
5. Fill in referral details:
   - Description
   - Priority
   - Supporting documents
6. Click "Submit Referral"

### Tracking Referral Status

1. Go to "Referrals" dashboard
2. View referral list with statuses:
   - Pending
   - Processing
   - Completed
   - Cancelled
3. Click on a referral for full details

### Updating Referral Progress

1. Open the referral
2. Click "Add Update"
3. Enter progress notes
4. Change status if needed
5. Click "Save Update"

## Chapter Management

### For Chapter Coordinators

#### Managing Members

1. Go to "Chapters" → Your Chapter
2. Click "Members" tab
3. Add new members:
   - Click "Add Member"
   - Search for constituent
   - Set membership status
4. Remove members:
   - Click member name
   - Click "Remove from Chapter"

#### Planning Activities

1. Go to "Chapter Activities"
2. Click "Schedule Activity"
3. Fill in details:
   - Activity name
   - Date and time
   - Location
   - Description
4. Click "Create Activity"

#### Tracking Attendance

1. Open the activity
2. Click "Take Attendance"
3. Check members present
4. Add notes if needed
5. Click "Save Attendance"

## Documents

### Uploading Documents

1. Go to relevant section (Referral, Constituent, etc.)
2. Click "Upload Document"
3. Choose file:
   - Supported formats: PDF, JPG, PNG, DOC
   - Maximum size: 10MB
4. Add description
5. Click "Upload"

### Managing Documents

1. View document list
2. Click document to preview
3. Options:
   - Download
   - Update description
   - Delete (if authorized)

### Document Versioning

- System tracks all versions
- Upload new version:
  1. Click existing document
  2. Click "Upload New Version"
  3. Select new file
  4. Add version notes

## Reports

### Viewing Reports

1. Click "Reports" in main menu
2. Choose report type:
   - Executive Dashboard
   - Operational Reports
   - Chapter Performance
   - Custom Reports

### Generating Custom Reports

1. Go to "Custom Reports"
2. Select report type
3. Set parameters:
   - Date range
   - Filters
   - Format (PDF, CSV)
4. Click "Generate Report"

### Exporting Data

1. Any list view has export option
2. Click "Export" button
3. Choose format:
   - CSV for spreadsheets
   - PDF for printing
4. File downloads automatically

## Search Features

### Global Search

1. Use search bar at top of page
2. Enter keywords
3. Results show across all modules
4. Click result to view details

### Advanced Search

1. Click "Advanced Search"
2. Select module to search
3. Apply filters:
   - Date ranges
   - Status
   - Categories
   - Custom fields
4. Click "Search"

### Saving Searches

1. After searching, click "Save Search"
2. Name your search
3. Access saved searches from profile
4. Set as default if needed

## Notifications

### Email Notifications

You'll receive emails for:
- New referral assignments
- Status updates
- Document uploads
- System announcements

### In-App Notifications

1. Bell icon shows notification count
2. Click to view notifications
3. Mark as read/unread
4. Click notification to go to item

### Notification Preferences

1. Go to "Settings" → "Notifications"
2. Choose notification types
3. Select delivery methods:
   - Email
   - In-app
   - SMS (if available)
4. Save preferences

## FAQ

### Q: How do I reset my password?
A: Click "Forgot Password" on login page, enter your email, and follow instructions.

### Q: Can I access the system on mobile?
A: Yes, the website is mobile-responsive and works on smartphones and tablets.

### Q: How do I report a bug?
A: Use the "Report Issue" link in footer or contact support.

### Q: What browsers are supported?
A: Chrome, Firefox, Safari, and Edge (latest versions).

### Q: How often is data backed up?
A: Data is backed up daily, with real-time sync to Notion.
